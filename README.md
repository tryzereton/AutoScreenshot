# Auto Screenshot
Otomatik Ekran Görüntüsü Kaydedici

You can use msbuild to compile:

### Automatic Screenshot for Windows

```cmd
msbuild AutoScreenshot.sln
```

or

```cmd
msbuild AutoScreenshot.csrpoj
```
